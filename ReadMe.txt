My Online Resume
This project is a personal online resume (Curriculum Vitae) built using standard web technologies. It serves as a professional portfolio to showcase my skills, work experience, educational background, and personal interests in a clean, responsive, and easily accessible format.

✨ Features
Clean and Responsive Design: Built with Bootstrap for a modern look that adapts well to various screen sizes (desktop, tablet, mobile).

Navigation: A fixed sidebar navigation allows for quick jumps between different sections of the resume.

Sections Included:

About: A brief professional summary and contact information.

Experience: Detailed breakdown of professional roles and responsibilities.

Education: Academic qualifications and achievements.

Skills: Highlighted technical skills and workflow proficiencies.

Interests: Personal interests that provide a glimpse into personality.

Languages: Overview of language proficiencies.

Social Links: Integration with social media profiles (e.g., LinkedIn) for easy access.

Font Awesome Icons: Used for visually representing skills and social media.

💻 Technologies Used
HTML5: For structuring the content.

CSS3: For styling, primarily using the Start Bootstrap theme's styles.css which is based on Bootstrap.

JavaScript: For interactive elements, particularly the smooth scrolling navigation.

Bootstrap 5.2.3: A popular CSS framework for responsive design and components.

Font Awesome 6.3.0: For scalable vector icons.

Google Fonts: For custom typography (Saira Extra Condensed and Muli).

🚀 Setup and Usage
To view this resume locally, follow these simple steps:

Clone or Download: Get a copy of this project to your local machine.

Navigate: Open the project folder.

Open index.html: Double-click the index.html file in your web browser (e.g., Chrome, Firefox, Edge).

The resume will open directly in your browser, and you can navigate through the sections using the sidebar.

✍️ Customization
Content: All textual content can be directly edited within index.html.

Styling: For major style changes, modify css/styles.css. For quick adjustments, inline styles can be added (though generally not recommended for large-scale changes).

Images: Replace assets/img/profile.jpg with your own profile picture. Ensure the new image has the same name or update the src attribute in index.html.

Links: Update the href attributes for social media icons, email, and navigation links.
